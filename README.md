# 🎼 OpusLog · 古典乐赏析与演出数字档案库

> **“典雅与序位的数字记录。”**  
> OpusLog 是一个基于 Hugo 构建的轻量化个人古典乐数字档案库。在这里，算法与流量被过滤，只留下对伟大作品的有意识倾听、名盘版本的碰撞，以及不可复制的现场观演回忆。

---

## ✨ 核心特色

- 🏛️ **仪表盘式档案大厅**：首页集成全局归档计数器，支持 **[全部] / [💿 唱片] / [🎟️ 现场]** 原地无刷新筛选，以及按时间与评分的多维排序。
- 💿 **名盘演绎版本矩阵**：专为唱片赏析设计的横向对比组件，直观呈现不同指挥、乐团、厂牌与版本的风格特质与推荐度。
- 🎟️ **现场观演时空凭证**：专属观演元数据面板，记录演出音乐厅、指挥/乐团、黄金座席坐标以及具体的声场体验。
- 🎼 **结构化古典乐元数据**：完美兼容作品编号（BWV / K. / Op.）、音乐时期（巴洛克 / 古典 / 浪漫 / 印象派）与作曲家标签索引。
- 🎨 **人文复古视觉设计**：采用温润的羊皮纸背景色（`#FDFBF7`）与勃艮第红（`#8B263E`）主题色，搭配矢量高音谱号专属角标。
